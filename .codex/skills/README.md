# Local Skills

Project-specific Codex skills for `tt.xima.work`.

- `tt-xima-content-centers` - safe work with the `centers` collection, MDX entries, and list/detail data shape.
- `tt-xima-ui-edits` - narrow Astro/Tailwind UI edits without drifting from the current interface language.
- `tt-xima-design-review` - audit component consistency, token reuse, and cross-surface design drift.
- `tt-xima-seo-mdx` - edit SEO, shared metadata, MDX frontmatter, canonicals, robots, OG, and JSON-LD safely.
- `tt-xima-list-data-hygiene` - normalize and review center list data, location fields, and filter contracts.
- `tt-xima-content-import` - convert raw center data, dumps, or spreadsheets into valid `centers` MDX entries.
- `tt-xima-release-checks` - run focused pre-ship checks for imports, collection sync, metadata, filters, and release info.

These skills are local to the repository and are meant to reduce repeated project discovery.
